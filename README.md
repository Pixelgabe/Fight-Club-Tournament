# Fight Club Tournament

A fun, interactive web-based tournament app inspired by the movie *Fight Club*. Pit your favorite images against each other in a single-elimination bracket to determine the ultimate champion!

## Features

- **Upload Images**: Select multiple images (fighters, memes, characters, etc.) to compete in the tournament.
- **Single-Elimination Bracket**: Automatic round progression from quarterfinals to the final match.
- **Visual Feedback**: See thumbnails of all participants and track the champion.

## How to Use

1. Open the app in your web browser.
2. Click the file input to upload at least 2 images.
3. Watch as the tournament begins with randomized matchups.
4. Click on the image you want to advance in each round.
5. Continue until a champion is crowned!

## Installation & Running

### Prerequisites
- Python 3.x
- Flask

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Pixelgabe/Fight-Club-Tournament.git
   cd Fight-Club-Tournament
   ```

2. Install Flask:
   ```bash
   pip install flask
   ```

3. Run the app:
   ```bash
   python fightclub_web.py
   ```

4. Open your browser and go to `http://localhost:5000`

## Technologies Used

- **Backend**: Python with Flask
- **Frontend**: HTML, CSS, JavaScript
- **Styling**: Custom CSS with CSS Variables for theming
